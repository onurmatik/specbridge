# Layout Reference

## App Shell
- Left: 80px sidebar
- Top: full-width header inside the main content column
- Body: page-specific content on `gray-50`

## Workspace
- Three-column visual rhythm: stream, divider, structured spec
- Stream column fixed-width and dense
- Spec column centered with generous vertical rhythm

## Dashboard
- Four-up metrics row
- Main content split into 8-column analysis area and 4-column status area

## History
- Left version timeline rail
- Right diff viewer with dual-column scrollable compare surface

## Handoff
- Left export-format rail
- Middle configuration panel
- Right document preview canvas
- Bottom export-history strip

## Gap Pages
- Use standard shell and panels
- Keep typography and chip language identical to drafted surfaces
