# Route Map

- `/`
  - Project Directory
- `/projects/<slug>/workspace/`
  - Align workspace / structured spec editor
- `/projects/<slug>/dashboard/`
  - Team alignment dashboard
- `/projects/<slug>/decisions/`
  - Decision timeline
- `/projects/<slug>/history/`
  - Spec version history and diff
- `/projects/<slug>/handoff/`
  - Export and handoff workspace
- `/projects/<slug>/assumptions/`
  - Assumptions register
- `/projects/<slug>/members/`
  - Members and roles
- `/api/projects/<slug>/...`
  - JSON actions/read models for stream, sections, assumptions, decisions, agents, exports, memberships
