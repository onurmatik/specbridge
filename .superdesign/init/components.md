# UI Components Inventory

- `GlobalSidebar`
  - Slim icon rail with active item styling and bottom user avatar.
- `PageHeader`
  - Breadcrumb, project title, status pill, active-member count, unresolved count, CTA actions.
- `MetricCard`
  - Dashboard card with heading, icon chip, large metric, muted supporting copy.
- `WorkspacePanel`
  - White card surface with border and low-elevation shadow.
- `StatusBadge`
  - Small uppercase pill with icon and contextual color.
- `TimelineCard`
  - Rounded decision/history card with timeline dot, status chip, metadata, and actions.
- `DiffRow`
  - Side-by-side previous/current content row with add/modify/remove tinting.
- `ExportHistoryTable`
  - Low-contrast history table with format chips and status pills.
- `MemberCard`
  - Avatar, role, title, email, and collaboration action area.
- `AssumptionCard`
  - Status-led record with impact badge, linked section, and validate/invalidate actions.
